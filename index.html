What is This Project About?

Feelsify.io is a web that allows users to record their daily levels of anxiety, stress, and depression. Users will also be able to include brief notes that might be relevant to how they are feeling. The app will store this information and provide users with easy-to-read graphs that will display trends in their mood. This app is ideal for users who want to stay on top of their mental health and be able to recognize any prolonged, negative trends of anxiety, stress, or depression, which can be detrimental to their health and well-being. In the future, we will add quizzes that hope to help users determine whether or not they might be experiencing anxiety, stress, depression, and even burn out.

Tech Stacks

Frontend:

  • React - client UI
  • Tailwind - CSS
  • Redux - Global state management (specifically react redux toolkit)
  • Yup - Client side form validation
  • Axios - http client for react

Backend:

  • ExpressJS - - Server
  • Bcrypt - Password hashing
  • MongoDB (?)

Other Tools for Development

  • Postman - For API debugging
  • Redux devTools plugin for firefox or chrome - For state management debugging
  • React developer tools plugin for firefox or chrome - For react debugging
